Chương trình viết bằng C++ sử dụng Windows API để khóa tạm thời máy tính người dùng.

⚙️ Chức năng chính
⏱️ Delay 10 giây trước khi chạy
🔐 Giải mã chuỗi bằng XOR
🚫 Khóa chuột và bàn phím (BlockInput)
🖥️ Hiển thị cửa sổ full màn hình (luôn trên cùng)
❌ Vô hiệu hóa Task Manager qua Windows Registry
🔁 Tự động chạy lại khi khởi động máy (Registry Run)
⏳ Khóa hệ thống trong 15 phút
🧹 Tự xóa file sau khi hoàn tất
